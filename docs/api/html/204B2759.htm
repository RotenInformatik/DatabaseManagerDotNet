Uninitialized0 The database is not initialized or has been closed.
ReadyNew1 The database is initialized and ready for use, using the newest known/supported version.
ReadyOld2 The database is initialized and ready for use, using an older known/supported version.
ReadyUnknown3 The database is initialized and ready for use, using an unknown/unsupported version (therefore, upgrading is not available).
New4 The database is initialized but is not ready for use because it is not available/created and requires an upgrade to create the database.
Unavailable5 The database is initialized but is not ready for use because it is not available/accessible.
TooNew6 The database is initialized but is not ready for use because its version is newer than the newest version known/supported.
TooOld7 The database is initialized but is not ready for use because its version is too old and cannot be upgraded.
DamagedOrInvalid8 The database is initialized but is not ready for use because it is damaged or invalid.
