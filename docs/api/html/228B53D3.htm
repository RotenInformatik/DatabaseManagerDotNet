Remarks

Assemblies is the list of assemblies used to lookup callbacks.

AssemblyCallbackBatchLocatorTConnection, TTransaction, TParameterTypes searches the assemblies for non-abstract class types which implement ICallbackBatchTConnection, TTransaction, TParameterTypes and which have a public parameterless constructor. Each found type is considered an independent batch. Each type such a batch is executed, a new instance of the corresponding type is instantiated (using CreateInstance(Type, Boolean)) and Execute(TConnection, TTransaction, IDbBatchCommandParameterCollectionTParameterTypes, String, Exception) is called.

By default, the name of the types which implement ICallbackBatchTConnection, TTransaction, TParameterTypes are used as the batch names and DontCare is used as transaction requirement. This can be overriden with the optional CallbackBatchAttribute, applied to the type, which overrides the name and/or transaction requirement.

Thread Safety
Static members of this type are not safe for multi-threaded operations. Instance members of this type are not safe for multi-threaded operations.
