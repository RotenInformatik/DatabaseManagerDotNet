Returns the state of the database. Can return null to let manager (or the implementation of IDbManager respectively) determine the state based on version alone.
version
Type: SystemInt32
Returns the version of the database. Return -1 to indicate when the database is damaged or in an invalid state. Return 0 to indicate that the database does not yet exist and needs to be created.

Return Value

Type: Boolean
true if the state and version of the database could be successfully determined, false otherwise. Details about failures should be written to logs.
