Remarks
 Important
See NameFormat for details about how script resources are searched in assemblies.

Assemblies is the list of assemblies used to lookup scripts.

Encoding is the used encoding for reading the scripts as strings from the assembly resources.

Currently, the following options are supported which are extracted from the scripts (see OptionsFormat for more details):

  • TransactionRequirement [optional] One of the DbBatchTransactionRequirement values (as string), e.g. /* DBMANAGER:TransactionRequirement=Disallowed */
 Note
If the TransactionRequirement option is not specified, DontCare is used.
Thread Safety
Static members of this type are not safe for multi-threaded operations. Instance members of this type are not safe for multi-threaded operations.
