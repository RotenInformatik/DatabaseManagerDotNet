Remarks
 Notes to Implementers
It is recommended that database backup creators implementations use this base class as it already implements most of the database-independent logic defined by IDbBackupCreator and IDbBackupCreatorTConnection, TTransaction, TParameterTypes.
Thread Safety
Static members of this type are not safe for multi-threaded operations. Instance members of this type are not safe for multi-threaded operations.
