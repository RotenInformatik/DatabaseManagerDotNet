Remarks

Database version detectors are used to detect the state and version of a database. What the version detection does in detail depends on the database type and the implementation of IDbVersionDetector.

Implementations of IDbVersionDetector are always specific for a particular type of database (or particular implementation of IDbManager respectively).

 Note
Database version detectors are mandatory. A database version detector must be configured before a database manager can be built or used.
 Note
IDbVersionDetector implementations are used by database managers. Do not use IDbVersionDetector implementations directly.
