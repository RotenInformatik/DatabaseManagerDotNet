Remarks

This boilerplate implementation uses upgrade steps provided by batches according to their names. GetBatchNamePattern must be implemented in derived types to provide a RegEx pattern which filters the batch names and extracts the source version numbers from each name. See GetBatchNamePattern for additional information.

Thread Safety
Static members of this type are not safe for multi-threaded operations. Instance members of this type are not safe for multi-threaded operations.
