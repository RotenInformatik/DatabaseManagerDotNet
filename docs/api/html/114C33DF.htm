Remarks

Database cleanup processors are used to cleanup a database. What the cleanup does in detail depends on the database type and the implementation of IDbCleanupProcessor but is usually something like "vacuum", recreate indices, etc.

Implementations of IDbCleanupProcessor are always specific for a particular type of database (or particular implementation of IDbManager respectively).

 Note
Database cleanup processors are optional. If not configured, cleanup is not available / not supported.
 Note
IDbCleanupProcessor implementations are used by database managers. Do not use IDbCleanupProcessor implementations directly.
