Gets whether the database manager supports the backup functionality.

Syntax

Copy
bool SupportsBackup { get; }

Property Value

Type: Boolean
true if the database manager supports backup, false otherwise.
Remarks
 Notes to Implementers
SupportsBackup should be set during the construction of the database manager and then remain unchanged and independent of the current database state or version (only depending on the database manager configuration).
