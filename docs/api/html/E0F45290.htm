Remarks

Database version upgraders are used to upgrade a databases version from version n to version n+1. What the upgrade does in detail depends on the database type and the implementation of IDbVersionUpgrader.

Implementations of IDbVersionUpgrader are always specific for a particular type of database (or particular implementation of IDbManager respectively).

IDbVersionUpgrader performs upgrades incrementally through multiple calls to Upgrade(IDbManager, Int32). Upgrade(IDbManager, Int32) is always called for the current/source version and then upgrades to the current/source version + 1. Therefore, database managers must call Upgrade(IDbManager, Int32) as many times as necessary to upgrade incrementally from the current version to the desired version.

 Note
Database version upgraders are optional. If not configured, upgrading is not available / not supported and some versioning information about the database will not be available.
 Note
IDbVersionUpgrader implementations are used by database managers. Do not use IDbVersionUpgrader implementations directly.
