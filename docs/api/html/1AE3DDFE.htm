IDbManagerUpgrade Method

Performs an upgrade to a specific database target version using the configured IDbVersionUpgrader.

Namespace:  RI.DatabaseManager.Manager
Assembly:  RI.DatabaseManager.Common (in RI.DatabaseManager.Common.dll) Version: 0.1.0.0
Syntax
C#
Copy
bool Upgrade(
	int version
)

Parameters

version
Type: SystemInt32
The target version to upgrade the database to.

Return Value

Type: Boolean
true if the upgrade was successful, false otherwise. Details about failures should be written to logs.
Exceptions
ExceptionCondition
ArgumentOutOfRangeExceptionversion is less than MinVersion, greater than MaxVersion, or less than Version.
InvalidOperationException The database is not in a ready or the new state.
NotSupportedException Upgrading is not supported by the database manager or no IDbVersionUpgrader is configured.
Remarks
 Notes to Implementers
State and Version are updated to reflect the state and version of the database after upgrade.
 Notes to Implementers
If version is the same as Version, nothing should be done.
 Notes to Implementers
Upgrading is to be performed incrementally, upgrading from n to n+1 until the desired target version, as specified by version, is reached.
