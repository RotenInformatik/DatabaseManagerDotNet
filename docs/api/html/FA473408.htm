ExceptionCondition

Remarks
 Notes to Implementers
State and Version are updated to reflect the state and version of the database after upgrade.
 Notes to Implementers
If version is the same as Version, nothing should be done.
 Notes to Implementers
Upgrading is to be performed incrementally, upgrading from n to n+1 until the desired target version, as specified by version, is reached.
