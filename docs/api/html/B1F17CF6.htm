IDbManagerBackup Method

Performs a backup using the configured IDbBackupCreator.

Namespace:  RI.DatabaseManager.Manager
Assembly:  RI.DatabaseManager.Common (in RI.DatabaseManager.Common.dll) Version: 0.1.0.0
Syntax
C#
Copy
bool Backup(
	Object backupTarget
)

Parameters

backupTarget
Type: SystemObject
The backup creator specific object which abstracts the backup target (e.g. a stream or a path to a file).

Return Value

Type: Boolean
true if the backup was successful, false otherwise. Details about failures should be written to logs.
Exceptions
ExceptionCondition
ArgumentNullExceptionbackupTarget is null.
ArgumentExceptionbackupTarget is of a type which is not supported by the configured IDbBackupCreator.
InvalidOperationException The database is not initialized.
NotSupportedException Backup is not supported by the database manager or no IDbBackupCreator is configured.
Remarks
 Notes to Implementers
State and Version are updated to reflect the state and version of the database after backup.
