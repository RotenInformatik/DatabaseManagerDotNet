Remarks

AggregateBatchLocatorTConnection, TTransaction, TParameterTypes is both a IDbBatchLocator and IListT implementation. It can dynamically combine multiple script locators and present it as one, doing lookup of scripts in the order of the list.

Thread Safety
Static members of this type are not safe for multi-threaded operations. Instance members of this type are not safe for multi-threaded operations.
