Remarks
 Notes to Implementers
The default value is (/\*\s*DBMANAGER:)(?<key>.+?)(=)(?<value>.+?)(\s*\*/) .
 Important
The options format must be a regular expression which provides two named captures: key and value . Those captures are used to extract key/value pairs from the script itself (e.g. SQL code). Using the default value, /* DBMANAGER:MyValue=123 would provide a key MyValue with the value 123 .
