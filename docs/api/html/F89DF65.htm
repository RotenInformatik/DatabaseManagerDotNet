Parameters

name
Type: SystemString
The name of the batch.
commandSeparator
Type: SystemString
The string which is used as the separator to separate commands within the batch or null if the batch locators default separators are to be used.

Return Value

Type: IDbBatch
The batch or null if the batch of the specified name could not be found. Details about failures should be written to logs.
